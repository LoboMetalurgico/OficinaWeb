# Oficina Web
Aplicativo base da Oficina de CRUD WEB.

## Sobre
Olá! Eu sou o Hugo, atualmente, sou graduando em Ciências da Computação pela UNIP. Realizo trabalhos como freelancer desde os 16 anos, com foco em desenvolvimento de softwares backend. Hoje, estou atuando como fullstack em uma pequena empresa.

Criei esta oficina originalmente para a Semana Tecnológica 2024, e decidi disponibilizá-la sob domínio público para que mais pessoas possam se beneficiar do conteúdo.

Este conteúdo, apesar de extremamente simples, cobre os conceitos básicos de um CRUD (Create, Read, Update, Delete) utilizando Node.js e Express. É ideal para iniciantes que desejam aprender sobre desenvolvimento web.

Sinta-se à vontade para explorar, modificar e compartilhar o projeto!

Se tiver alguma dúvida ou sugestão, não hesite em entrar em contato comigo!

## Tecnologias Utilizadas
- Node.js
- Express
- MariaDB (MySQL)
- PromiseORM
